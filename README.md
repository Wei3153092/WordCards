📇 WordCards 
這是一個互動式英文單字卡應用程式。

主畫面：支援單字顯示、音標、解釋與自動播放功能

編輯畫面：可即時修改單字內容並同步存檔

✨ 核心功能 
自訂單字庫載入：支援讀取 .txt 格式的 TSV 單字檔，包含單字、音標、音檔路徑與中文解釋/字根字首拆解。

多媒體發音整合：結合 Windows Media Player (WMPLib) 元件，支援點擊單字自動播放對應的 .mp3 語音檔。

自動播放模式 (Auto-Play)：內建 Timer 計時器，可一鍵啟動自動輪播模式，系統將自動間隔切換單字並發音。

快捷鍵操控：

Enter：切換至下一個單字。

Space (空白鍵)：重新播放當前單字發音。

即時編輯與持久化儲存：雙擊單字清單可開啟編輯視窗 (frmEditWord)。修改後的內容會即時更新於 UI，並直接覆寫回原始的文字檔中。
